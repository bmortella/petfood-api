# petfood-api
Este repositório contém a API do aplicativo petFood para o Laboratório de Engenharia de Software.

# Instalar dependencias  
```npm install```  
  
# Inicializar servidor  
### Modo desenvolvimento  
```npm run dev```  
### Modo produção
```npm run start```
  
# Testes
```npm run test```  
  
# Lint
```npm run lint```
